3D Gaussian Splatting Viewer
This is a Three.js-based implemetation of a renderer for 3D Gaussian Splatting for Real-Time Radiance Field Rendering, a technique for generating 3D scenes from 2D images. Their project is CUDA-based and needs to run natively on your machine. This page is based on Mark Kellogg's web viewer on github https://github.com/mkkellogg/GaussianSplats3D.
